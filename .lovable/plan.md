

# Сайт-визитка ведущего праздников

Современный одностраничный сайт-визитка с админ-панелью для управления контентом.

## Публичная часть (лендинг)

### 1. Главный экран (Hero)
Яркий баннер с фото ведущего, именем, слоганом и кнопкой «Оставить заявку». Современный чистый дизайн с акцентными цветами.

### 2. Обо мне
Блок с рассказом о ведущем: опыт, стиль работы, фото. Краткая биография, которая вызывает доверие.

### 3. Услуги и цены
Карточки по 6 категориям:
- 💍 Свадьбы
- 🏢 Корпоративы
- 🎂 Юбилеи
- 🎓 Выпускные
- 🎧 DJ + Звук
- 🎉 Любой праздник

Каждая карточка — описание программы и стоимость (или «от ...»).

### 4. Фото и видео галерея
Галерея с фотографиями с мероприятий, возможность просмотра в полноэкранном режиме. Ссылки на видео.

### 5. Отзывы клиентов
Карусель с отзывами: имя клиента, текст отзыва, тип мероприятия.

### 6. Форма заявки / Контакты
Форма обратной связи (имя, телефон, тип мероприятия, дата, комментарий). Ссылки на мессенджеры (WhatsApp, Telegram), телефон, соцсети.

## Админ-панель (Lovable Cloud)

Закрытый раздел для ведущего, доступный по логину/паролю:
- Управление услугами и ценами
- Добавление/редактирование отзывов
- Загрузка фото в галерею
- Просмотр заявок от клиентов
- Редактирование текста «Обо мне» и контактной информации

## Технические детали
- База данных через Lovable Cloud (Supabase) для хранения контента
- Аутентификация для входа в админку
- Адаптивный дизайн (мобильные + десктоп)
- Современный минималистичный стиль с тёплыми акцентами
